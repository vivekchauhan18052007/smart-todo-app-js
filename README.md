# Smart Todo App

A simple and interactive Todo App built using HTML, CSS, and JavaScript.

## Features
- Add, Edit, Delete Tasks
- Mark tasks as completed
- Data saved using LocalStorage
- Persistent tasks after reload

## Tech Stack
- HTML
- CSS
- JavaScript (DOM + LocalStorage)
